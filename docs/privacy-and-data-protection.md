## IP Summary

RSU Nexus mainly uses original interface designs, source code, documentation, validation data, screenshots, and project branding created by the Quartet team.

The project also uses external platforms and tools, including Figma, OpenAI Codex or ChatGPT, Supabase, Vercel, GitHub, and Microsoft Power BI. These tools are used under their respective service terms.

The homepage hero photograph is an external licensed asset from Wikimedia Commons and must retain its creator, source, licence, and modification attribution.

The team must not use the official Rangsit University logo without permission and must not present RSU Nexus as an official or university-endorsed system.

User-uploaded listing images, profile images, and written content create the main intellectual-property risk. Users should be required to upload only content they own or have permission to use. The prototype should provide moderation rules for misleading, copied, inappropriate, or prohibited content.

AI-assisted design, coding, and documentation support must be reviewed, understood, tested, and edited by the project team before submission. AI-generated output should not be treated as automatically correct or original.
