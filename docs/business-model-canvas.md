## Project Title
Campus Resource Sharing Platform

## 1. Customer Segments
- Students
- Campus clubs and organizations
- University administrators

## 2. Value Proposition
Provides a single platform for students to share, browse, and reserve campus resources easily.

## 3. Channels
- Web application
- University announcements and social media

## 4. Customer Relationships
- User accounts
- Notifications for bookings and requests
- Feedback/contact support

## 5. Revenue Streams / Sustainability Logic
This is a non-profit university project. It provides operational value by improving resource sharing, reducing waste, and increasing resource accessibility.

## 6. Key Resources
- Web application
- Database
- Hosting
- Development team

## 7. Key Activities
- Manage resource listings
- Process reservations
- Update resource availability
- Admin management

## 8. Key Partners
- University administration
- Student organizations
- Resource owners

## 9. Cost Structure
- Web hosting
- Database storage
- Development and maintenance time

## Final Note
This business model supports the MVP features, including resource listing, search and filter, reservation requests, resource details, admin management, and dashboard summary.
