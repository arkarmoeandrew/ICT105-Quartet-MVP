## Reflection

### Which feature creates the strongest user value?
Browse and search resources because it helps users quickly find the resources they need.

### Which feature creates the strongest business or operational value?
Admin management because it keeps resource information accurate and maintains system reliability.

### Which required feature is still weak or unclear?
Dashboard analytics may need additional metrics to provide more meaningful insights about resource usage and reservations.
