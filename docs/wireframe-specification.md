## Wireframe Quality Rules
- Use consistent navigation, colors, buttons, and labels across all screens.
- Avoid decorative-only screens without functional purpose.
- Display realistic sample resource listings instead of empty placeholders.
- Ensure every wireframe directly supports the corresponding system requirements.
- Wireframes may be created using Figma, Canva, PowerPoint, HTML/CSS, or another suitable prototyping tool.
